# AI Website Copy Generator for Local Businesses

## Business Chosen
Cafe Corner – Bangalore

## About the Project
Many local cafes struggle with unclear website content, weak value propositions,
and confusing calls-to-action. As a result, potential customers often leave
without understanding what the cafe offers or why they should visit.

This project demonstrates how Prompt Engineering can be used to generate
clear, engaging, and conversion-focused website copy for local businesses
using AI tools.

## Objective
The objective of this task is to design a structured prompt system that can:
- Generate homepage copy with a strong value proposition
- Create service and menu descriptions tailored to a cafe business
- Write persuasive call-to-action (CTA) sections
- Adapt tone and language for a local neighborhood cafe
- Produce content that is ready to be published on a real website

## Business Details
- Name: Cafe Corner
- Location: Bangalore
- Type: Neighborhood Cafe
- Target Audience: Students, working professionals, casual visitors
- USP: Freshly brewed coffee, cozy ambience, affordable pricing

## Tools Used
- ChatGPT

## Prompt Engineering Approach
Structured and reusable prompts were designed to:
- Control tone (warm, friendly, welcoming)
- Ensure business-specific language
- Avoid generic AI-style content
- Focus on clarity, trust, and conversions

Separate prompts were created for:
- Homepage content
- Services / menu page content
- Call-to-action sections

## Deliverables
The project includes:
- `prompts.md` – Structured prompts used to generate website copy
- `outputs.md` – AI-generated website copy including:
  - Homepage content
  - Services / menu descriptions
  - Call-to-action sections

## Outcome
The generated website copy is clear, natural, and business-focused.
It is suitable for direct use on a real cafe website and demonstrates
practical prompt engineering skills for real-world client scenarios.

## Conclusion
This project showcases how AI-powered prompt engineering can help local
businesses create effective website content without the need for expensive
copywriting services.
